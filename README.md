## Installation

1 - Clone the repo 

```bash
git clone https://github.com/Tripper98/FACTO_N.git
cd FACTO_N
```


2 - Install the package

```bash
pip install .
```
3 - To use the CLI

```bash
pip install .[cli]
```

## Usage 

```bash
python mathsup -n NUMBER 
# NUMBER = Provide the number for which you want to calculate the factorial, n 
```
